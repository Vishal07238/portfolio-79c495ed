

# Vishal Arkalwar — Portfolio Website

A stunning dark-themed portfolio with cyan, magenta, and gold accents, glassmorphic elements, and smooth animations — built as a single-page React app.

---

## 1. Theme & Design System
- **Dark background** with cyan (`#00d4ff`), magenta (`#ff2d7b`), and gold (`#d4af37`) accent colors
- Glassmorphic card styling with subtle borders and backdrop blur
- Custom animations: fade-in reveals, hover scale effects, shimmer accents
- Google Fonts: Space Grotesk (headings), IBM Plex Sans (body), IBM Plex Mono (code/labels)

## 2. Floating Navigation
- Centered floating navbar with blur background: **Dev** logo + Home, About, Skills, Projects, Contact
- Smooth scroll to each section
- Mobile hamburger menu

## 3. Hero Section
- Asymmetrical two-column layout
- **Left**: Profile image from Cloudinary with cyan glow border and decorative corner accents
- **Right**: "FULL-STACK AI ENGINEER" label, large "Vishal Arkalwar" heading, stats row (6+ Projects, 5 Tech Stacks, End-to-End Solutions), bio paragraph, and two CTA buttons ("Explore Work" + "Let's Connect")
- Scroll indicator at bottom

## 4. About Section
- "WHO I AM" label with "Data-Driven Problem Solver" heading (gradient text)
- Left column: bio paragraphs about full-stack expertise
- Right column: 3 interactive expandable cards — Full-Stack Solutions, Data-Driven Insights, Cloud Infrastructure — with gradient backgrounds
- Stats row: 6+ Projects Built, 5 Tech Stacks, 100% Commitment, Full Stack Dev

## 5. Skills Section
- "TECHNICAL ARSENAL" label with "Skills & Expertise" heading
- 4 skill cards in a grid, each with a distinct accent color border:
  - **Data Analysis** (cyan): Python, SQL, Pandas, NumPy, Matplotlib, Power BI
  - **Data Engineering** (magenta): ETL/ELT, Airflow, MySQL, Snowflake, AWS
  - **Frontend Development** (purple): HTML, CSS, JavaScript
  - **Backend Development** (gold): Node.js, Express.js, PostgreSQL
- Proficiency level cards below: Advanced, Intermediate, Foundation with skill tags
- Hover effects with color-shifted glows

## 6. Projects Section
- "PORTFOLIO HIGHLIGHTS" label with "Featured Projects" heading
- 6 project cards in alternating 2-column layout (card left / icon right, then flipped)
- Each card shows category badge, title, short description
- **Click to open detailed modal** with:
  - Full project description (STAR format or multi-section)
  - Tech stack display
  - GitHub / Live Demo buttons where applicable
- Projects: Customer Behaviour, Crop Price Prediction, Olist AWS Pipeline, Smart Kisan, Volunteer Management System, Service Backend
- "More projects coming soon" banner with Visit GitHub button

## 7. Contact Section — "Get In Touch"
- Left side: Email, Location, Response Time info cards + social links (GitHub, LinkedIn, Email)
- Right side: Contact form (Name, Email, Subject, Message) with per-field focus colors
- **Formspree integration** (ID: mzdadzee) for email delivery
- Form validation, loading state, success/error toast notifications, auto-clear on success

## 8. Footer
- 3-column layout: Brand info, Navigation links, Connect links
- GitHub icon in bottom left
- Gradient divider line
